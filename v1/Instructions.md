Este é o projeto MISplica. 

Trata-se de uma página interativa simples que objetiva auxiliar o ensino-aprendizagem do MIS, método da engenharia semiótica. Quero fazer uma rodada de refatoração nele, para melhorar estrutura de código, usabilidade e acessibilidade.

---

# Regras básicas
Seguem algumas regras:
- Não mexa nos estilos que criei, nem crie novos por enquanto ou a não ser que eu solicite.
- Quero atualizar a funcionalidade de raio-x utilizando *anchor positioning* e _position-try_ para melhor posicionar os popovers. Não se preocupe com compatibilidade, eles já estão disponíveis para a baseline desde de 2025.
- Quero reduzir ao máximo a necessidade de scripts que possam ser substituídos por funcionalidades modernas de CSS.
- Quero usar propriedes customizadas para determinar as cores
